# 백엔드 개발
### 백엔드 개발 이란?
- 사용자 인터페이스를 만들어내는 코드 작성과 관련없는 모든 개발
- API, 라이브러리 제작 등등

### 백엔드 개발자는 어떤 일을 하는가?
- 웹 개발에서 백엔드 개발자는 대체로 어플리케이션 작동 로직을 만드는 일에 관여
- 시스템 아키텍처에 개입
- 알고리즘을 적용하고 문제를 해결

### 백엔드 개발 기본 기술
- SQL, 데이터베이스
- 기술 스택에 맞는 서버 측 언어 사용
- 애플리케이션 아키텍처
- 인프라

# DBA와 데브옵스
### DBA
- 데이터베이스 유지보수
- 최적화 (성능 분석, 인덱싱)

### 데브옵스
- 코드를 개발해서 생성하기까지 종합적인 과정을 혼자 담당
- 빌드한 코드를 자동으로 테스트하고 품질을 확인
- 소프트웨어 개발자는 엄밀히 말해 데브옵스에 속한다

### 데브옵스의 업무
- 코드를 개발에서 생산까지 보내는데 쓰이는 절차와 도구를 알아야함(IDE, 빌드 방법 등)
- 소스를 제어하는 방법 (GIT)
- 테스트에 대한 기초 지식, 테스트 자동화
- 애플리케이션 패키징, 배포 절차
- 스테이징 서버, 프로덕션 서버에서 환경 설정을 관리하는 방법
- 모니터링 하는 방법
